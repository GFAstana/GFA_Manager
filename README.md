# GFA Manager

Система управления тренировками HEMA-клуба GFA Astana.

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Данные (мок)

- **Клиенты** — имя, контакт, дата заведения
- **Тренеры** — ФИО
- **Слоты** — время, направление, зал, дефолтный тренер
- **Абонементы** — название, стоимость, кол-во тренировок, срок действия
- **Оплаты** — клиент, тариф, сумма, период, использовано, статус
- **Посещения** — дата, слот, привязка к абонементу
- **Тренировки** — слот + дата, список атлетов, отметивших посещение
- **Новая тренировка** — `/training/new` — выбор слота, даты и атлетов по активным абонементам
- **Профиль клиента** — `/clients/[id]` — сводка, абонемент, оплаты, посещения
- **Профиль тренера** — `/trainers/[id]` — слоты, направления, проведённые тренировки

Данные лежат в `src/lib/data/` и пока хранятся в памяти.

Мок-данные ориентированы на **2026-06-08** (понедельник) — на эту дату 8 атлетов с активным абонементом и свободные слоты 10:00 / 18:30.
